English Vocabulary learning web app

* Java Spring MVC
* Database : Heidi SQL(Maria DB)

1. Database : DB 입력, 저장,\n 불러오기
2. Security : 로그인, 회원가입
* 구성
1. Main Page : Navigation bar
2. Study : Youtube video, Writing Memo
3. Memo : Memo list, Writing memo
4. Inquiry : Inquiry list, Writing inquiry
